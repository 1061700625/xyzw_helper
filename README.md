# xyzw_helper

> - 刚开始搞，欢迎感兴趣的大佬一起来开发~    
> - 对于电脑上的安卓模拟器，推荐使用“**夜神模拟器**”！    
> - 需要修改分辨率1280x720,320dpi。如果直接在真机上运行，由于分辨率差异可能会导致问题。如果改不了，可以装**光速虚拟机**。如果有电脑，就装**夜神模拟器**。
